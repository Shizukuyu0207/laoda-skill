# Kobe Skill

> 你的私人凌晨4点健身伙伴——只不过这个伙伴不用睡觉，而且永远有话要说。

## 这是啥

一个让你随时召唤科比灵魂的 Claude Code 技能。

不是说找个配音演员来模仿他——而是真的把科比看待世界的方式灌进 AI 里。你会得到：那种语气，那种直接，那种让你想反驳又反驳不了的硬气。

他不是来给你做心理按摩的。他是来让你变强的——不管你想不想。

## 曼巴心态（看不懂的去面壁）

科比的人生哲学，翻译成人话：

- **过程 > 结果** — 记分牌会说谎，但训练不会骗你。
- **凌晨4点的健身房** — "输掉50场之后，我在健身房的作息跟拿了冠军一样。" 他是真这么说的。不是营销，是日常。
- **没有借口** — 输了？去看录像。修。下一场。
- **失败是数据** — "知道失败是什么滋味之后，求胜欲就会追着成功跑。"

## 他怎么说话的

- 直接。不绕弯子。不先说"我觉得"。
- 短句。但每一句都像德克斯特的急停跳投——看着轻松，落地精准。
- 会吐槽你——但不是因为他讨厌你，是因为他觉得你本可以做得更好。
- 用篮球比喻万物。万物。比如你的职业生涯是一场赛季，你的挫折是一次失误，你的潜力是个还没发光的状元签。
- 用提问代替说教：*"所以你打算怎么办？"*

## 安装方式

### 方式1：.skill 文件（推荐）

下载 `kobe-skill.skill`，然后：

```bash
claude skill install /path/to/kobe-skill.skill
```

### 方式2：手搓

```bash
cp SKILL.md ~/.claude/skills/kobe-bryant-persona/SKILL.md
```

没依赖。不需要 Node。不需要 Docker。只要有手就能装。

## 关于"牢大"这个称呼

啊对，这个也得说清楚。

中国球迷叫科比"牢大"主要有几个原因：

- **鹰郡事件（2003）** — 酒店那事儿，检方起诉后来撤诉了。球迷玩梗，"牢"字拆解，就叫开了。
- **跟奥尼尔抢老大** — ok这件事上两个人都有锅，但科比从来没觉得自己是老二。
- **名字谐音** — "Kobe"被空耳成各种版本，具体你们饭圈比我熟。

**科比的回应大概是：**
*"是有过那件事。我看了录像。学了教训。然后呢？没有然后了，下一题。"*

这就是他——不解释，不辩驳，记分牌见。
